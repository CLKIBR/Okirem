# Blog Modülü

## Amaç
Kullanıcıların ve editörlerin güncel haber, duyuru ve eğitim içeriklerini paylaşabileceği blog altyapısı sağlar.

## Bileşenler
- `BlogListComponent`: Blog yazılarını listeler.

## Servisler
- `BlogService`: Blog yazılarını API üzerinden çeker.

## Dosya Yapısı
- `blog-list.ts`: Component mantığı
- `blog-list.html`: Template
- `blog-list.scss`: Stil
- `blog.service.ts`: Servis
- `blog.module.ts`: Modül
- `blog.md`: Dokümantasyon

## Geliştirilebilir Özellikler
- Blog yazısı ekleme, düzenleme, silme, arama, etiketleme
- Yorumlar, beğeni, moderasyon

---
Bu dosya, mimari plana %100 bağlı olarak otomatik oluşturulmuştur.
