# Sosyal Modül

Bu modül, kullanıcıların platformda birbirleriyle sosyal etkileşimde bulunmasını sağlar.

## Özellikler
- Yorum, beğeni, paylaşım gibi sosyal aktiviteler
- Kullanıcı etkileşimlerinin listelenmesi
- Topluluk yönetimi

## Kullanım
- `SocialListComponent` ile etkileşimler listelenir.
- `SocialService` ile sosyal veriler alınır.

## Mimari
- Angular component/service yapısı
- Responsive arayüz
- Genişletilebilir sosyal API entegrasyonu
