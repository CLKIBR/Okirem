# Gelişmiş Raporlama Modülü

Bu modül, platformdaki gelişmiş raporları ve analizleri sunar.

## Özellikler
- Kullanıcı ve kurs bazlı raporlar
- Detaylı analiz ve görselleştirme
- Yöneticiler için özel raporlar

## Kullanım
- `AdvancedReportListComponent` ile raporlar listelenir.
- `AdvancedReportService` ile rapor verileri alınır.

## Mimari
- Angular component/service yapısı
- Responsive arayüz
- Genişletilebilir raporlama API entegrasyonu
