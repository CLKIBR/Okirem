# Mobil Modül

Bu modül, platformun mobil cihazlarda kullanılabilirliğini ve mobil özelliklerini yönetir.

## Özellikler
- Push bildirim, offline mod, mobil arayüz
- Mobil özelliklerin listelenmesi ve yönetimi
- Kullanıcıya özel mobil deneyim

## Kullanım
- `MobileListComponent` ile mobil özellikler listelenir.
- `MobileService` ile özellik verileri alınır.

## Mimari
- Angular component/service yapısı
- Responsive arayüz
- Genişletilebilir mobil API entegrasyonu
