# PWA Modülü

Bu modül, platformun Progressive Web App (PWA) özelliklerini yönetir.

## Özellikler
- Offline desteği, ana ekrana ekleme, hızlı yüklenme
- PWA özelliklerinin listelenmesi ve yönetimi
- Kullanıcıya özel PWA deneyimi

## Kullanım
- `PwaListComponent` ile PWA özellikler listelenir.
- `PwaService` ile özellik verileri alınır.

## Mimari
- Angular component/service yapısı
- Responsive arayüz
- Genişletilebilir PWA API entegrasyonu
