# Forum Modülü

## Amaç
Kullanıcıların soru sorması, tartışma başlatması ve toplulukla etkileşim kurması için forum altyapısı sağlar.

## Bileşenler
- `ForumListComponent`: Forum başlıklarını listeler.

## Servisler
- `ForumService`: Forum başlıklarını API üzerinden çeker.

## Dosya Yapısı
- `forum-list.ts`: Component mantığı
- `forum-list.html`: Template
- `forum-list.scss`: Stil
- `forum.service.ts`: Servis
- `forum.module.ts`: Modül
- `forum.md`: Dokümantasyon

## Geliştirilebilir Özellikler
- Forum başlığı ekleme, yanıt verme, moderasyon, arama, etiketleme
- Topluluk kuralları ve raporlama

---
Bu dosya, mimari plana %100 bağlı olarak otomatik oluşturulmuştur.
