# Donanım Entegrasyonu Modülü

Bu modül, eğitim platformunun çeşitli donanım cihazlarıyla entegrasyonunu sağlar.

## Özellikler
- Akıllı tahta, tablet, mobil cihaz entegrasyonu
- Cihaz listesi ve yönetimi
- Donanım tabanlı eğitim senaryoları

## Kullanım
- `DeviceIntegrationListComponent` ile cihazlar listelenir.
- `DeviceIntegrationService` ile cihaz verileri alınır.

## Mimari
- Angular component/service yapısı
- Responsive arayüz
- Genişletilebilir donanım API entegrasyonu
